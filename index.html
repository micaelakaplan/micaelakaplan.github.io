Movie MadLibs

Movie MadLibs is an NER driven MadLibs generator, trained on a custom annotated set of data from OMDB. The blanks are the result of running the text through a CRF model and having all entities appear as blanks with the label given by the model.

Fluency Friend

Fluency friend is an Alexa and Google Assistant skill whose goal is to teach beginning Spanish speakers basic commands. Ask in English, it'll give you Spansih. Use Spanish incorrectly, it'll correct you!
